# Portfolio
Det här projektet är min personliga portfolio-sida. Den fungerar som en samlingsplats för de webbprojekt jag utvecklar och fungerar även som ett live-exempel på hur jag skriver kod.

## Teknisk info
Sidan är byggd med React och Vite. Jag använder TypeScript för hela projektet och CSS Modules för att hålla stylingen lokal till varje komponent. Projektet körs och deployas via Cloudflare Workers.
